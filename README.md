How to use:

* first check out openelec git repository
* then copy this repo contents over there
* or just use make-repo.sh to build all addons

## Binary Addon Respository ##
All addons are compiled for x86_64 and i386.
They are  available on my openelec community repository http://code.google.com/p/openelec-addons/


## Addons maintained by romanlum

### SABnzbd-Suite-german ###

German Version of SABnzbd-Suite
This addon uses the german version of sickbeard (https://github.com/cytec/Sick-Beard)

### mupen64plus ###

N64 Emulator for openelec
